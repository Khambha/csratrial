AnuVikar

Application Information

This is the AnuVikar application. It allows you to create and process JSON files based on the provided form parameters. Follow the steps below:

  1. Fill in the form fields with the required information.
  2. Click the "Browse" button to select the output folder.
  3. Click the "Create JSON" button to generate the JSON file.
  4. Wait for the application to finish processing the data.
  5. The processed data will be saved in the selected output folder.
  6. Click the "Terminate" button to stop the ongoing process.
